ARD module
==========

Reports on the Apple Remote Desktop information fields. 

Retrieves information from /Library/Preferences/com.apple.RemoteDesktop.

The following information is stored in the table:

* id - Unique ID
* serial_number
* text1 - Text field 1
* text2 - Text field 2
* text3 - Text field 3
* text4 - Text field 4
